aniTrace
========

一次追了很多部動畫，卻常常忘記動畫看到哪一集嗎？讓 aniTrace 幫你輕鬆解決動畫紀錄的問題。


安裝
----

1. git clone
1. vim application/config/database.php  # 設定資料庫帳號密碼
1. vim application/config/config.php    # 設定網站相關參數
1. mysql db_name -u user -p password < db_schema.sql  # 建立資料表
